# Explore Nepal 🇳🇵

AI-powered tourism and exploration platform built with Next.js, TypeScript, Tailwind CSS, and full stack technologies.

## ✨ Features

- Modern responsive UI
- Smooth animations
- AI-powered experiences
- Full stack architecture
- Destination discovery
- SEO optimized
- Mobile friendly
- Scalable structure

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Node.js
- Prisma ORM
- PostgreSQL
- Framer Motion

## 🚀 Run Locally

```bash
npm install
npm run dev
```
